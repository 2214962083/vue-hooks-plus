## ✨ 特性

- 易学易用
- 支持 SSR
- 丰富的 Hooks
- 覆盖大部分业务场景
- 使用 TypeScript 构建，提供完整的类型定义文件
- 支持按需加载，减少打包体积

## 📦 安装

```
$ npm install --save vue-hooks-plus
# or
$ yarn add vue-hooks-plus
```

## 🔨 使用

```typescript
import { useRequest } from 'vue-hooks-plus'

// or 按需引入

import useRequest from 'vue-hooks-plus/es/useRequest'
```

## 🏃 启动文档

hooks 目录下

```
npm run initial
npm run docs:dev
```
