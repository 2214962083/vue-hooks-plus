## API

```typescript
const { container } = usePreview(target)

```

### Params

| 参数   | 说明             | 类型                                                        |
| ------ | ---------------- | ----------------------------------------------------------- |
| target | DOM 节点或者 Ref | `VueComponent` \| `string` \| `JSX.Element` |

### Result

| 参数   | 说明             | 类型                                                        |
| ------ | ---------------- | ----------------------------------------------------------- |
| container | DOM 节点或者 Ref | `Element` \| `() => Element` \| `JSX.Element` |
