# Vue3-hooks-plus 安装

## 📦 安装

```
$ npm install --save vue3-hooks-plus
# or
$ yarn add vue3-hooks-plus
```

## 🔨 使用

```typescript
import { useRequest } from 'vue3-hooks-plus'
```

<br />

::: warning
不兼容 vue2 版本
:::
